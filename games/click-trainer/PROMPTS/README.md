# Click Trainer - Development Prompts

This folder contains the prompts used to create the Click Trainer game with AI assistance.

## Prompt Files

- **[development.md](development.md)** - The main prompt used to generate the complete game

## How This Game Was Built

The Click Trainer was built using a single comprehensive prompt that specified:
- Core mechanics (click targets, reaction timing)
- Configurable options (ball count, size)
- Statistics tracking (average, best time)
- UI/UX requirements (full-screen canvas, glow effects)

## Using These Prompts

You can use these prompts to:
1. Recreate the game from scratch
2. Understand the design decisions
3. Learn prompt engineering for game development
4. Extend the game with new features
